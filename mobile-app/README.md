# Barcody Mobile App

This is the Expo mobile application for Barcody.

## Setup

- **Framework**: Expo (React Native)
- **Routing**: expo-router
- **Language**: TypeScript
- **Theme**: Dark Mode (Neon Blue)

## Project Structure

- `app/(tabs)/`: Tab navigation screens (Scan, History, Settings).
- `constants/Colors.ts`: Theme colors.
- `app.json`: Configuration.
- `eas.json`: Build profiles.

## Assets

Place your design assets in `assets/images/`:
- `icon.png`: 1024x1024
- `adaptive-icon.png`: 1024x1024
- `splash-icon.png`: 200x200 (or larger)

## Running

```bash
npm install
npx expo start
```
